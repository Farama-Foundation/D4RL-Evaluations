# D4RL Evaluations

This repository contains the algorithms used to evaluate tasks in the [D4RL benchmark](https://github.com/Farama-Foundation/D4RL) from the original paper. All code is lightly modified from other public repositories on Github. This repo currently is not being maintained or updated to support the new changes to the D4RL repository.

## Reference Codebases.

- [AlgaeDICE](https://github.com/google-research/google-research/tree/master/algae_dice)
- [BRAC, BC, BEAR](https://github.com/google-research/google-research/tree/master/behavior_regularized_offline_rl)
- [AWR](https://github.com/xbpeng/awr)
- [BCQ](https://github.com/sfujim/BCQ)
- [Continuous REM](https://github.com/theSparta/off_policy_mujoco)
